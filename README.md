Web Technologies Lab 5_Lab 6

In this lab, we studied and implemented different CSS layout techniques and responsive design concepts. The main focus was on understanding how web pages are structured using CSS properties such as display, float, positioning, Flexbox, and CSS Grid. We also learned how to create responsive designs using media queries for different screen sizes.

Throughout the lab, we performed multiple tasks including working with display properties (block, inline, inline-block), creating float-based layouts, and applying different positioning techniques like fixed, absolute, and sticky. We also built modern layouts using Flexbox and CSS Grid, such as navigation bars, card layouts, dashboards, and the holy grail layout using different methods.

In addition, we implemented responsive designs like a responsive data table, mobile-first layouts, and media query-based adjustments for various devices. This lab helped us understand how to combine multiple CSS techniques to create flexible and user-friendly web designs.

Overall, this lab provided practical experience in building modern, responsive web layouts and improved our understanding of CSS design techniques.
